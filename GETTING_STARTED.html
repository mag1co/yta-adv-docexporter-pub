1. Install the app

Install from YouTrack Marketplace or upload manually to your YouTrack instance.

2. Attach to a project

Go to YouTrack Admin → Apps → Adv.DocExporter and attach it to the desired project. No additional configuration required.

3. Export

Open any Knowledge Base article — four new menu items will appear in the article options menu (⋯):

  • Export to XLSX — Tables with formatting
  • Export to XLSX (no formatting) — Tables as plain text
  • Export to DOCX — Full article with formatting
  • Export to DOCX (no formatting) — Full article as plain text

Note: Due to a YouTrack issue, correct icons for menu items are only displayed in YouTrack 2026.1 and later. In earlier versions, a single generic app icon is shown instead.

The file downloads automatically.

4. Settings (optional)

The app works out of the box with sensible defaults. If needed, you can customize project-level settings in Your Project → Apps → Adv.DocExporter → Settings:

  • Default File Name Prefix — prefix for exported files (default: table-export)
  • Add Date to File Name — append current date to file name (default: on)
  • Notification Mode — popup (show result in widget) or alert (YouTrack toast)
